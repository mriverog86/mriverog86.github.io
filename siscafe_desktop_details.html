SisCafé

  • Category: Desktop application
  • Date: Jun 2016 - 2019
  • Associated with: Information Technology and Communications Company of the Cuban Ministry of Agriculture (EICMA)
  • Role performed: Programmer, giving technical support and adding new features.
  • Technologies:
    • Java PostgreSQL JasperReports

Description

  • System for the management of coffee production, used by all the coffee producing companies of the Cuban Ministry of Agriculture Agroforestry Group (GAF). It controls all the processes: Agrotechnics, Production and Harvest, providing abundant reports.
  • Desktop application developed in Java using PostgreSQL as database and JasperReports for reporting. It has been already replaced by a new version of the application developed using Symfony.
